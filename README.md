# Homomorphic Encryption

> Privacy-preserving computation on encrypted data.

## Overview

This repository contains implementations, experiments, notes, and resources related to Homomorphic Encryption (HE) and Fully Homomorphic Encryption (FHE).

Homomorphic Encryption enables computations to be performed directly on encrypted data without decrypting it, ensuring data privacy and secure computation.

---

## Features

- Fundamental concepts of Homomorphic Encryption
- BFV, CKKS, and TFHE schemes
- Practical encryption/decryption examples
- Privacy-preserving machine learning demos
- Research references and learning resources
